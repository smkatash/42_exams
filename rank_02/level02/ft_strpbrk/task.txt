Assignment name	: ft_strpbrk
Expected files	: ft_strpbrk.c
Allowed functions: None
---------------------------------------------------------------

Reproduce exactly the behavior of the function strpbrk
(man strpbrk).

The function should be prototyped as follows:

char	*ft_strpbrk(const char *s1, const char *s2);


strpbrk -- locate multiple characters in string

DESCRIPTION
     The strpbrk() function locates in the null-terminated string 
     s the first occurrence of any character in the string charset 
     and returns a pointer to
     this character.  If no characters from charset occur anywhere 
     qin s strpbrk() returns NULL.
